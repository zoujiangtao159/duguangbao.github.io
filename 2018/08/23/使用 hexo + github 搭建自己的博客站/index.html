安装nodejs

Node.js 是一个基于 Chrome V8 引擎的 JavaScript 运行环境，简单的说就是运行在服务端的 JavaScript，更深的东西请慢慢挖掘

git的大名想来都听过，它现已成为了世界上最先进的分布式版本控制系统之一，这里使用它还可以快速的将你的本地资料发布到github

git的安装也十分简单，下载安装，一路next就好，这里给出网址：https://git-scm.com/downloads
